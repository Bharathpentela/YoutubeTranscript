Title: QuickTube: Instant Summaries
by @bharathpentela
Description:
QuickTube is a web application designed to provide instant summaries of YouTube videos. 
By extracting the transcript of a given YouTube video, QuickTube leverages Google's Gemini AI 
to generate concise and informative summaries, saving users time and allowing them to quickly 
grasp the key points of any video.

Features:

YouTube Transcript Extraction:
 Extracts transcripts from YouTube videos if subtitles are available.
AI-Powered Summaries: 
Uses Google Gemini to generate brief, meaningful summaries of video content.
Thumbnail Display:
 Shows the video's thumbnail to ensure users have the right video.

User-Friendly Interface: 
Simple and intuitive interface built with Streamlit.

Technologies Used:

Python: Core programming language.
Streamlit: For building the web application interface.
YouTubeTranscriptApi: To fetch video transcripts.
Google Gemini AI: For generating summaries from transcripts.
dotenv: For managing environment variables securely.
